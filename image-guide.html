Guide

Save your images with the exact filenames listed below and place them in the same folder as moodboard.html. The site auto-detects each file and drops it into the correct slot — no code editing needed. If an image isn't found, a clean labeled placeholder shows in its place.

Back to Moodboard

How it works

Every image slot in the moodboard has a hidden <img> tag pointing to a specific filename. When that file exists in the same folder, it loads automatically and fills the slot at full bleed. Both .jpg and .png formats are supported. For logos, use a transparent .png. Landscape images work best for wide/hero slots; portrait for tall mood slots. There's no need to refresh the page after dropping files in — just open or reload the moodboard.
